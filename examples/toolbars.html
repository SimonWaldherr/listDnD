Works as expected in Safari, reasonably well in Firefox, and less than adequately in IE.

Firefox 1.0 isn't respecting the ":active" pseudo-selector on the list elements. When you mousedown on a button it should look pressed in, but in Firefox it doesn't. Presumably this is because we capture 'onmousedown'. A workaround should be possible in JavaScript.

These examples are really just extensions of DnD Sortable Lists.

These toolbars represent a rough proof-of-concept. More like a "Golly gee, look what I can do" than an actually suggestion for how to do something. So in this case, golly gee, you can rearrange the toolbar buttons...

 
 

The horizontal toolbar is built from an identical list as the vertical toolbar. Different CSS styles make the difference.
